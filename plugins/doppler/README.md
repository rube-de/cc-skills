# doppler

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Claude Code](https://img.shields.io/badge/Claude%20Code-Plugin-purple.svg)]()

Doppler secrets management plugin for Claude Code. Provides comprehensive assistance with the Doppler CLI, project/config management, secrets injection, CI/CD integrations, and security best practices.

## Installation

```bash
claude plugin install doppler@rube-cc-skills
```

## Usage

Invoke the skill with `/doppler` or ask about Doppler-related topics:

- "Set up Doppler for this project"
- "Inject secrets into my Docker container"
- "How do I sync secrets to AWS Secrets Manager?"
- "List all secrets for the staging config"
- "Set up service tokens for CI/CD"

## Features

- Complete CLI command reference
- Secrets injection patterns (env vars, file mount, templates)
- Project/config/environment hierarchy management
- Integration guides for AWS, GCP, Azure, Vercel, GitHub Actions, Docker, Kubernetes
- Service token management for CI/CD
- Security best practices and troubleshooting
